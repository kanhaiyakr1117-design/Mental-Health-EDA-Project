# 📊 Mental Health EDA using ETM (Excessive Technology Model)

## 📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of mental health in relation to technology usage using the ETM (Excessive Technology Model). The goal is to analyze how screen time, sleep, and digital activities influence stress levels.

---

## 🎯 Objectives
- Analyze relationship between screen time and stress
- Study impact of sleep on mental health
- Identify patterns in digital behavior
- Perform multivariate analysis

---

## 🛠️ Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Dataset
The dataset contains:
- Screen Time Hours
- Sleep Hours
- Stress Level
- Social Media Usage
- Gaming Hours
- Total Usage

---

## 📊 Key Analysis Performed
- Correlation Heatmap
- Scatter Plot (Multivariate Analysis)
- KDE Plot (Distribution Analysis)
- Feature Engineering (High Screen Time, Sleep Category)

---

## 🔍 Key Insights
- Weak correlation observed between most variables
- No strong linear relationship between screen time and stress
- User behavior appears independent and complex
- Sleep patterns do not strongly correlate with usage

---

## ⚠️ Conclusion
The analysis suggests that mental health is influenced by multiple factors and cannot be explained by a single variable like screen time. More advanced modeling may be required for deeper insights.
